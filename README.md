# 🎓 智能选课系统 (Smart Course Selector)

## 🌟 项目愿景 (Our Vision)
- **中文：** 解决学校选课期间服务器拥堵、响应慢的问题，让每位同学都能公平、快速地选到心仪课程。
- **English:** To solve the problem of server congestion and slow response during school course selection, ensuring every student can choose their desired courses fairly and quickly.

## 💡 核心功能 (Key Features)
- **🚀 极速响应 (High Responsiveness):** - **CN:** 优化数据传输逻辑，减少抢课延迟。
  - **EN:** Optimize data transfer logic to minimize registration latency.
- **📊 实时名额 (Real-time Availability):** - **CN:** 动态更新并显示课程剩余名额。
  - **EN:** Dynamically update and display remaining course slots.
- **🛡️ 智能排冲突 (Auto-Conflict Detection):** - **CN:** 自动检测所选课程的时间是否重叠。
  - **EN:** Automatically detect time overlaps between selected courses.

## 🛠️ 技术思路 (Technical Logic)
1. **轻量化数据 (Lightweight Data):**
   - **CN:** 使用 JSON 格式存储课程信息，减少服务器负担。
   - **EN:** Use JSON format to store course info, reducing server load.
2. **极简前端 (Minimalist Frontend):**
   - **CN:** 界面简洁化，确保在弱网环境下也能快速加载。
   - **EN:** Simplify the UI to ensure fast loading even under poor network conditions.
